# ohkwon718.github.io

Simple GitHub Pages site for `ohkwon718.github.io`.

Project screenshots/GIFs live in **`assets/previews/`** (copied from repos so the site does not depend on `raw.githubusercontent.com`). See that folder’s `README.md` for the source map.

## Local preview

You can open `index.html` directly in a browser, or run a quick local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Publish

1. Commit and push to the `main` branch of this repository.
2. In GitHub, open `Settings` -> `Pages`.
3. Set source to `Deploy from a branch`, branch `main`, folder `/ (root)`.
4. Your site will be available at `https://ohkwon718.github.io`.
