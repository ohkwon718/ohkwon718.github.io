# Project preview assets

These files are **local copies** so the Pages site loads previews from the same origin (stable, no hotlink dependency).

| File | Used for |
|------|-----------|
| `cali360.gif` | cali360 |
| `bayesian-mvpose.gif` | Bayesian Multi-View Pose Tracking |
| `toy-gs.png` | Gaussian Splatting (toy_GS) |
| `open3d-app-template.gif` | Open3D App Template |
| `multicam-calibration.png` | Camera Calibration Tools (multicam-calibration / calib-with-mp4) |
| `Distributed_Kalman_Filter.gif` | Distributed Kalman Filter |

**When a preview changes:** replace the file here and commit.

**Adding more:** drop `your-name.ext` in this folder, then set `src="assets/previews/your-name.ext"` in `index.html`.

Optional: compress with `oxipng`, `gifsicle`, etc. before committing.
